Disciplinas do Curso de Informática

Disciplinas do Curso

  • Programação Orientada a Objetos

    Esta disciplina cobre os fundamentos da programação orientada a objetos (POO), incluindo conceitos como classes, objetos, herança e polimorfismo. Os alunos aprendem a criar sistemas robustos e reutilizáveis usando esses princípios.

  • Redes de Computadores

    Em Redes de Computadores, os alunos exploram os princípios básicos das redes de computadores, incluindo protocolos, topologias, e a configuração e gerenciamento de redes locais (LAN) e redes amplas (WAN).

  • Banco de Dados

    Esta disciplina aborda o design e a administração de bancos de dados relacionais, incluindo a modelagem de dados, a linguagem SQL e a implementação de sistemas de gerenciamento de bancos de dados.

  • Programação Web

    Desenvolvimento Web envolve a criação de sites e aplicações web, cobrindo tecnologias como HTML, CSS, JavaScript e frameworks modernos. Os alunos aprendem a construir interfaces de usuário interativas e responsivas.

  • Manutenção de Computadores

    Esta disciplina foca na manutenção e reparo de hardware de computadores, incluindo diagnóstico de problemas, substituição de componentes e técnicas de manutenção preventiva e corretiva.
